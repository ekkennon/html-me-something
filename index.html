My Programming Background:

QBASIC Visual Basic ITT Tech C++ Java for Android MO Tech BJC Software QA Engineer C Certification CS Degree LaunchCode (LC101)

QBASIC

Mid 90's

After installing DOS one day I was looking around to see if there was anything interesting and found something called QBASIC.EXE and remembered seeing a book laying around the house with the same title.


Found a con of using the old BASIC languages after writing a program with a timer based on the processor speed, because the processor I had was much faster than what was around when the book was written. My first taste of programming got me hooked.

Visual Basic

High School

This was the only programming language offered at my high school and it seemed like the natural next step for me at the time.


Came out of this with a better understanding of the basic concepts and an introduction to common mistakes (such as infinite loops).

ITT Tech

2001 - 2006

At the time tech schools seemed like a better option than traditional schools because they are more focused on the skills employers wanted to see.


After getting my Associate's degree and being halfway to getting my Bachelor's, looking at my debt and not getting interviews I got discouraged. Quit school and gave up programming.

C++

2010

A friend of mine got me interested in programming again. He would tell me about the GUI C++ applications he was writing. I got an itch to try it programming again and when I would ask for advice he always had suggestions.


I was writing all the code in Notepad++ and using makefiles to help with compiling. Toward the end of a year I was starting to use SDL and OpenGL for 2D and 3D graphics, but quickly realized this isn't a huge interest of mine.

Java for Android

2011

While working with C++ I would think of apps I would like to have for free on my iPod Touch. Due to the lower cost I decided to try writing Android apps.


This was a steep learning curve for me, having forgotten most of what I had learned about while at ITT Tech. I was familiar with some syntax as I had been writing in C++ but since the only thing I really understood about Object Oriented programming was that an object is a class and it is a way to separate out code, I did have trouble getting up to speed with reading Java. On top of that I was trying to understand the Android API. Because of this experience I have the ability to learn any languages, concepts, techniques, and API's that I put time into learning.

MO Tech

2012 - 2015

I heard an ad on the radio for this school. They had a very high job placement rate and I thought with my experiences in the past 2 years I would be able to show an employer that I have the skills they are looking for.


The school closed before I earned my degree but while there I was able to take more advanced classes than I had before. Design patterns theory, Java 3 with Spring and Hibernate, Microsoft ASP with the MVC pattern.

BJC

2012 - 2015

At this time, BJC HealthCare had something called the PRN Program. Made for people with little or no experience, a project manager could ask for an available PRN to work for a short period of time (up to 6 months) on a task that didn't require high level skills. The project manager saved money vs. hiring a contractor and the PRN got experience and networking opportunities. Many students from my school were in, or had been in, this program.


I worked with the testing (support) team for the only in-house made software at BJC. I gained experience in manual regression testing and was thrilled everytime I found a bug missed by others. After several failed attempts to remain with the testing or development team I moved to another project. My new supervisors, knowing I was in school for programming, gave me an opportunity to use my skills where I gained valuable experience to apply for an official programming job.

Software QA Engineer

2015 & 2016

This seemed like a great opportunity to gain skills, knowledge, and experience.


I did get out of this even more than I expected. However, that took place, in total, in about a quarter of my time in this role. Most of my time spent here was writing the same code repeatedly, when I had code to write. I realized the best way for me to get what I am looking for was to go back to school.

C Certification

2016

This is way of showing employers that I have skills. Risky because some see programming certifications as something for those who don't have the skills to write code but are good at memorizing or book learning. I am not good at either of those but was able to study hard and pass the test.


To me it isn't necessary to get as deep into a language for most programmers as a certification requires but it gave me an idea of what to expect if I decide to get a Java certification.

CS Degree

2017 to present

I would like to show potential employers that I have the skills to get the degree.


I am starting with the gen ed classes but have started looking at the Computer Science section on Khan Academy.

LaunchCode (LC101)

2017 Jan - Jun

This was suggested to me by a former co-worker with the idea that it will be easier for them to place me in an apprenticeship if I've taken a class.


Starting Unit 2 and everything up to now has been review. Soon we will get into things I've never done before.
